# OriginChaosAI
A chaotic system based on the Feigenbaum constant, combined with an LSTM neural network for time series prediction

## Core Constants
- Feigenbaum constant: δ ≈ 4.6692016091
- Period-doubling limit: r∞ ≈ 3.5699456718

## Dependencies
- numpy
- torch

## Usage
Run the main script to generate Feigenbaum chaotic sequences and train the LSTM model for prediction.

## Purpose
- Nonlinear dynamics research
- Chaotic time series modeling
- AI and chaos theory experiments
